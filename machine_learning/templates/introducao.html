# Do que se trata a aprendizagem de máquina(AM)?

  • Ramo da Inteligência Artificial
  • Solução de um problema aprendida a partir de dados (amostras ou exemplos).
  • Mínimo de intervenção humana.
  • Uso de técnicas de indução
  • Modelos cognitivos - Dados (commodity do século XXI) transformados em conhecimento.

# Quando deve ser utilizada?

  • O problema está bem claro? Representação adequada é crucial para o sucesso de um projeto.
  • O problema não tem uma solução convencional (determinística)? Treinar e avaliar modelos cognitivos NÃO é uma tarefa trivial.
  • Há dados suficientes em quantidade e qualidade? Representatividade das amostras de treinamento é primordial.
  • Há profissionais de AM disponíveis? Experiência no preparo dos dados, seleção da técnica adequada e definição de protocolo experimental robusto.
  • Há hardware adequado para uso das técnicas de aprendizado de máquina? Algumas técnicas demandam processamento paralelo.

# Quais as abordagens?

> Não supervisionada

Não requer dados rotulados.

> Supervisionada

Requer dados rotulados.

> Semi-supervisionada

Alternativa quando não temos dados rotulados em quantidade suficiente.

> Por reforço

Não precisa de dados de treinamento.
