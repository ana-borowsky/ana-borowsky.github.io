Conceitos Básicos

Fundamentos essenciais para compreender o funcionamento dos algoritmos de aprendizagem de máquina.

# O que é Machine Learning?

  • Campo da Inteligência Artificial focado em aprendizado a partir de dados
  • Modelos aprendem padrões sem serem explicitamente programados
  • Baseado em estatística, otimização e teoria da computação
  • Permite previsões, classificações e tomadas de decisão

# Elementos fundamentais

  • Dados: matéria-prima do aprendizado
  • Modelo: representação matemática do problema
  • Treinamento: processo de ajuste do modelo
  • Predição: uso do modelo para gerar respostas

# Tipos de dados

> Dados estruturados

Organizados em tabelas (ex: planilhas, bancos de dados).

> Dados não estruturados

Texto, imagens, áudio, vídeo.


# Processo de treinamento

  • Divisão dos dados em treino e teste
  • Aprendizado a partir dos dados de treino
  • Avaliação com dados não vistos
  • Ajuste de parâmetros (tuning)

# Problemas comuns

> Overfitting

Modelo aprende demais os dados de treino e perde generalização.

> Underfitting

Modelo simples demais, não captura padrões relevantes.


# Features (Atributos)

  • Variáveis usadas como entrada do modelo
  • Qualidade das features impacta diretamente o desempenho
  • Engenharia de atributos é etapa crítica

# Labels (Rótulos)

  • Resposta esperada (target)
  • Usado em aprendizado supervisionado
  • Exemplo: "spam" ou "não spam"

# Métricas de avaliação

> Acurácia

Percentual de acertos do modelo.

> Precisão

Quantos positivos previstos estão corretos.

> Recall

Quantos positivos reais foram encontrados.


# Pipeline de Machine Learning

  • Coleta de dados
  • Pré-processamento
  • Treinamento do modelo
  • Avaliação
  • Deploy (produção)
